# OneRepStrength Voice Commands

## How to Activate
- **Earbud button:** Tap play/pause on your earbuds
- **In-app:** Tap the microphone button

---

## Exercise Control

| Command | What it does |
|---------|--------------|
| **"Start [exercise name]"** | Start a specific exercise |
| **"Begin [exercise name]"** | Same as start |
| **"Do [exercise name]"** | Same as start |
| **"Let's do [exercise name]"** | Same as start |
| **"[Exercise name]"** | Just say the exercise name |

*Examples: "Start leg press", "Do chest press", "Lat pulldown"*

---

## Navigation

| Command | Alternatives | What it does |
|---------|--------------|--------------|
| **"Next"** | "Next exercise", "Next one", "Move on" | Go to next exercise |
| **"Skip"** | "Skip phase", "Skip this", "Next phase" | Skip current timer phase |
| **"Skip rest"** | "End rest", "No rest", "Ready" | End rest period early |

---

## Timer Control

| Command | Alternatives | What it does |
|---------|--------------|--------------|
| **"Pause"** | "Hold", "Wait", "Stop timer" | Pause the timer |
| **"Resume"** | "Continue", "Go", "Unpause", "Start timer" | Resume timer |

---

## Exercise Completion

| Command | Alternatives | What it does |
|---------|--------------|--------------|
| **"Done"** | "Finished", "Complete", "I'm done", "That's it" | Complete current exercise |
| **"Another set"** | "One more", "Again", "Repeat", "Do it again" | Repeat the same exercise |
| **"Stop"** | "Stop workout", "End workout", "Quit", "Finish workout" | End entire workout |

---

## Logging Weight

| Command | Examples |
|---------|----------|
| **"Log [number]"** | "Log 150", "Log 45" |
| **"[number] pounds"** | "150 pounds", "200 pounds" |

---

## Tips

1. **Speak clearly** - The app uses iOS speech recognition
2. **Wait for the beep** - A sound plays when listening starts
3. **Keep it simple** - Short commands work best
4. **Exercise names** - Use the names as they appear in your workout list
5. **Works with music** - Voice commands work while music is playing

---

## Troubleshooting

- **Mic not working?** Check Settings → Privacy → Microphone → OneRepStrength is ON
- **Speech not recognized?** Check Settings → Privacy → Speech Recognition → OneRepStrength is ON
- **Earbud button not working?** Make sure "Voice Commands" is enabled in app settings
